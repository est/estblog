什么是Planet-lab？什么是CoDeeN？

Posted 2007-03-26 | archive


Planet-lab，官方定义为An open platform for developing, deploying, and accessing planetary-scale services，翻译成中文就是：星球实验室，一个用于开发、部署和使用的“星系级别”的网络服务的开放平台。

PlanetLab

PlanetLab is a global research network that supports the development of new network services. Since the beginning of 2003, more than 1,000 researchers at top academic institutions and industrial research labs have used PlanetLab to develop new technologies for distributed storage, network mapping, peer-to-peer systems, distributed hash tables, and query processing.

PlanetLab currently consists of 760 nodes at 379 sites.

它是一个全球性的研究网络，用于新型网络服务的开发。自从2003年起，超过1,000过来自顶级学术机构和业界实验室在PlanetLab上开发了新技术包括：分布式存储、网络印射、P2P系统、DHT（分布式哈希表）和查询处理。PlanetLab目前有760个节点和379个站点。

 

A Content Distribution Network for PlanetLab

 

CoDeeN 包含下列服务：

  • CoBlitz, a scalable Web-based distribution service for large files.
    这个服务主要用来传输大文件。FedoraCore5/6 就是用 CoBlitz 来发布的。类似这个服务的使用方法很简单，在网址前面加上 http://coblitz.codeen.org:3125/ 就行了。例如我要下载 Nero Linux：
    http://ftp6.usw.nero.com/software/NeroLINUX/nerolinux-2.1.0.4b-x86.deb
    使用CoBlitz的网址就是：
    http://coblitz.codeen.org:3125/ftp6.usw.nero.com/software/NeroLINUX/nerolinux-2.1.0.4b-x86.deb
    我用迅雷下载NeroLinux几乎一直是 0% 不动，用CoBlitz一会儿就下载完了，速度在2KB～50KB之间（教育网）。
    可以到CoBlitz 上去查看 Fedora Core 各个发行版本。
  • CoDeploy, an efficient synchronization tool for PlanetLab slices.
    这个是用来在PlanetLab的切片主机之间同步的工具
  • CoDNS, a fast and reliable name lookup service.
    传说中的P2P的DNS服务。可以在编程里调用，也可以很方便的用telnet获得：

    • telnet localhost 4119
    Trying 127.0.0.1...
    Connected to localhost.
    Escape character is '^]'.
    www.princeton.edu      <- domain name
    128.112.128.15             <- IP address
    Connection closed by foreign host.

    也可以下载使用Linux/MacOS版本：CoDNS 0.52

  • CoTop, a command-line activity monitoring tool for PlanetLab.
    PlanetLab的监控工具
  • CoMon, a Web-based general node/slice monitor that monitors most PlanetLab nodes.
    PlanetLab节点的Web版监控工具
  • CoTest, a login debugging tool that tries to be human-friendly.
    登陆的友好调试器
  • CoViz, a visualization tool graphically displaying PlanetLab activity.
    可视化的PlanetLab活动显示。
