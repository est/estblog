javascript 节拍器

Posted 2024-09-26 | stdout

因为娃的节奏感糊成一坨了。加上码农特有的穷B气质，想去找个节拍器app；结果各大市场都是各种广告，怕给娃用不正经；

又找了很多公众号版的，小程序版的，网页版的，普遍都是广告满天飞。失望之余，想起来上次JS播放DTMF对 WebAudio 有点感觉，要不这次自己干脆再手搓一个。

调研了一圈发现这玩意没那么容易，主要就是javascript在浏览器里 setInterval 是有时间抖动（jitter）的，不能保证节奏精确。如何才能做到精准采样打节拍呢？WebAudio！

搜了一圈发现一位叫 Grant James 已经写好放在 github 了。那还等什么直接拿来～～呃等等，有如下需要：

  1. 无广告
  2. 去掉 .js .css 依赖，单文件更快
  3. 精简不必要的代码结构，用来魔改

改完之后发现两个坑也跟着一起修补了：

  1. 用 wakeLock 防止手机上使用息屏
  2. 突破固定的4拍子，加上可以增减的节拍控制

最后就放在 https://lab.est.im/metronome/ 这里了。有需要的拿去用。
